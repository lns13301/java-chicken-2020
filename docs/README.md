# 기능 구현

1. 출력
    - [ ] 메인화면
    - [ ] 기능 선택
    - [ ] 테이블목록
    - [ ] 테이블 선택
    - [ ] 메뉴 목록
    - [ ] 메뉴 수량 입력
    - [ ] 테이블 주문 내역
    - [ ] 결제 진행
    - [ ] 최종 결제 금액
    
2. 입력
    - [ ] 기능 번호
    - [ ] 테이블 번호
    - [ ] 메뉴 번호
    - [ ] 메뉴 수량
    
3. 테이블
    - [ ] 테이블 목록
    - [ ] 주문 테이블 목록
    
4. 포스기
    - [ ] 등록할 메뉴를 입력받고 테이블에 메뉴 추가
    - [ ] 결제
        - [ ] 치킨 류가 10개 넘는 경우 10,000원씩 할인 (10개 단위)
        - [ ] 현금 결제 시 최종 결제 금액에서 5% 추가 할인
    - [ ] 주문 혹은 결제가 불가능 할 경우 이유를 출력 후 다시 주문 가능 하도록 해야함
    - [ ] 최종 결제 금액 조회
